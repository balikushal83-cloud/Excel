# 📈 Excel Dashboard & Business Analytics Project

This repository contains Excel-based business analytics dashboards and reporting assets designed for data visualization, reporting, and business insights.

The project demonstrates how Excel can be used to create interactive dashboards, customer summaries, and categorized analytical reports for decision-making.
